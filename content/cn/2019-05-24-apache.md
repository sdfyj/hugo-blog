### 安装Apache服务器
安装
```
yum -y install httpd
```
启动apache服务器 
```
systemctl start httpd
```
设置开机自动启动 
```
systemctl enable httpd
```
关闭apache服务器 
```
systemctl stop httpd
```
网页存放目录为`/var/www/html/`,阿里云服务器要安全组开放80端口才能使用公网IP进行访问，
